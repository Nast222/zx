# Random Password Generator

**Автор:** [Лиза [Соколова]

## Описание программы

Это графическое приложение (*GUI*) для генерации надежных случайных паролей. Программа позволяет пользователю гибко настраивать параметры пароля: выбирать его длину с помощью ползунка и указывать, какие группы символов использовать (цифры, буквы, спецсимволы).

Все сгенерированные пароли автоматически сохраняются в историю, которую можно просмотреть в виде таблицы внутри приложения. Данные истории хранятся в локальном файле формата *JSON*. Проект использует систему контроля версий *Git* для отслеживания изменений в коде.

> **⚠️ Важное замечание по безопасности:** Пароли в истории хранятся в открытом (незашифрованном) виде. Не используйте это приложение для генерации паролей к критически важным сервисам (банки, госуслуги) без дополнительного шифрования файла `history.json`.

## Технологии

*   **Python:** Язык программирования.
*   **Tkinter:** Стандартная библиотека для создания графического интерфейса.
*   **Random:** Стандартная библиотека для генерации случайных значений.
*   **JSON:** Формат для сохранения истории паролей.
*   **Git:** Система контроля версий.

## Инструкция по запуску

1.  **Установите Python:**
    Убедитесь, что на вашем компьютере установлен Python версии 3.6 или выше. Библиотека `tkinter` обычно входит в стандартный комплект поставки Python.

2.  **Получите исходный код:**
    Скачайте папку с проектом или клонируйте репозиторий с помощью Git:
    ```bash
    git clone https://github.com/ваш_логин/название_репозитория.git
    ```

3.  **Запустите приложение:**
    Откройте терминал (или командную строку), перейдите в папку с проектом и выполните команду:
    ```bash
    python password_generator.py
    ```

4.  **Использование:**
    *   Установите нужную длину пароля с помощью ползунка.
    *   Отметьте галочками необходимые типы символов (рекомендуется использовать все).
    *   Нажмите кнопку «Сгенерировать пароль».
    *   Скопируйте пароль, нажав кнопку «Копировать».

## Примеры использования и тесты

1.  **Тест на минимальную длину:**
    *   *Действие:* Передвиньте ползунок длины в минимальное значение (обычно 8).
    *   *Ожидаемый результат:* Программа сгенерирует короткий, но сложный пароль.

2.  **Тест на отсутствие символов:**
    *   *Действие:* Снимите все галочки (Цифры, Буквы, Спецсимволы) и нажмите «Сгенерировать».
    *   *Ожидаемый результат:* Появится окно с ошибкой «Выберите хотя бы один тип символов».

3.  **Тест на историю:**
    *   *Действие:* Сгенерируйте 2-3 разных пароля.
    *   *Ожидаемый результат:* В таблице «История генераций» появятся новые строки с деталями настроек и временем создания.
